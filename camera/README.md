# "Photo Boo" camera
Takes a photo, thenreplaces a face with a background

Created for the 20Mission Ghosted (Party) at Public Works in San Francisco on Halloween

## Requirements
- dlib
- opencv
- pathlib2


```
$ pip install python-opencv
$ pip install dlib
$ pip install pathlib2
```

## Running

```
$ ./run_photoboo.py
```
